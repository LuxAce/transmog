Congratulations! Your Word document has been analyzed and any applicable automatic assignment have been applied! Make note of the current URL, this is the only place you can access this working document.

Now what?

You may scroll down to view the document. Each paragraph is represented in a rectangular box. Those that have been assigned a position in the resulting XML structure are represented in red

For each unassigned paragraph, you may:

  • Assign a type using the dropdown list
  • Drag it to another position within the document
  • Delete it

At any time you may download the current XML representation, or the underlying raw xml.
